## 📋 About This Project

This is a modern, responsive portfolio website for **Tiwa Adedeji**, a Virtual Assistant who helps busy founders stay organized through administrative and workflow support.

The site is designed to:
- Showcase Tiwa's services and expertise
- Build trust through testimonials and stats
- Convert visitors into consultations
- Present a professional, structured brand identity

---

## ✨ Features

- **Modern Aesthetic** — Clean, professional design with custom "notched corner" styling
- **Fully Responsive** — Optimized for all screen sizes (mobile, tablet, desktop)
- **Interactive Testimonial Carousel** — Auto-sliding client testimonials
- **Structured Service Showcase** — Clear presentation of 6 core services
- **Skills Grid** — Interactive tool and skills display with clickable pills
- **Direct Booking Integration** — Google Calendar appointment scheduling
- **WhatsApp & Email Contact** — Multiple communication channels
- **Social Media Optimization** — Open Graph tags for rich link previews

## 🔧 How to Customize

### Update Personal Information

1. **Profile Photo:** Replace `./assets/tiwa.PNG` with your own image
2. **Name & Title:** Update the `<title>` and `og:title` tags
3. **Email:** Update all `mailto:` links and contact details
4. **Social Links:** Update LinkedIn, Twitter, and WhatsApp URLs
5. **Booking Link:** Update the Google Calendar appointment URL

### Update Open Graph Tags

Edit these meta tags in `<head>` section of `index.html`:

```html
<meta property="og:title" content="Your Name · Your Title" />
<meta property="og:image" content="https://your-url.com/preview-image.PNG" />
<meta property="og:url" content="https://your-url.com/" />
